# INKWELL — Modern Blog Website

A clean, fully client-side blog web application built with vanilla HTML5, CSS3, and JavaScript. It includes a dark/light theme toggle, category filtering, live search, featured posts, a post reader view, and a like button — all with zero external JavaScript dependencies.

**Live Demo:** [blog-website-29d1.onrender.com](https://blog-website-29d1.onrender.com)

---

## Features

- **Dark / Light theme toggle** — switch between dark and light modes with a smooth transition
- **Category filtering** — filter blog posts by topic such as Technology, Design, Lifestyle, and News
- **Live search** — search posts in real time by title and excerpt
- **Featured post** — highlights one post using a wider featured card layout
- **Post reader view** — opens a full blog post without reloading the page
- **Like button** — toggleable heart button with live like count update
- **Responsive layout** — works smoothly on mobile, tablet, and desktop screens
- **Noise texture overlay** — adds a subtle visual texture to the background
- **Glassmorphism navbar** — sticky blurred navigation bar
- **Zero dependencies** — no frameworks, no build tools, and no npm required
